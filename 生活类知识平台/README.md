## 生活类知识平台

运用了thinkphp框架。网站命名为寻觅，标语是发现生活之美。是一个生活类知识平台网站，致力于帮助用户解决生活中的小烦恼，为用户带来便利的同时也增添了更多的生活乐趣。包含了热门推荐类、节日习俗类、家常菜谱类、旅游出行类、生活百科类、诗词类和娱乐类这七个类目。
